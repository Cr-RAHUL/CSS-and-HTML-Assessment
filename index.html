HyperText Markup Languagge(HTML) is the set of markup symbols or codes inserted in to a file intended for dislay on the internet. The markup tells web browsers how to display a web page's words and images.

CSS(Cascading Style Sheets) is used to style and layout webpages- for example,to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.

React is a free and open-source front-end JavaScript library fro building user interfaces based on UI components.It is maintained by Meta and a community of individual developers and compaines.

What is NodeJs used for?It is used for server-side programming, and primarily deployed for non-blocking, event-driven servers, such as traditional web sites and back end API services, but was originally designed with real-time.
